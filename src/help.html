Basic Operations

  • For basic operations, such as +,-,*,÷ use following syntax:
    • [FirstNumber] [Operator] [SecondNumber] and hit Enter (=).
    Output of the operation will be shown. Example: 6 + 2 = 8
  • Working with previous result.
    • To work with previous result, you can call ANS function, or simply
    continue with operation, without [FirstNumber] given.
    Example: + 4 = 12 (based on result from first paraghraph)
  • Working with signed numbers
    • To work with signed numbers, please use ± symbol on calculator button.
    It will automatically convert number into *(-1) form.
    Example: 2 ± = -2

Functions

  • In order to use special functions correctly, follow these instructions.
    • All the operations, except Power only accept one number as input.
    Correct order: [FirstNumber] [Operation] Maintain the given order.
    Example: "9" + "√" will result in operation √9 = 3
  • Power Function
    • To use the Power function two numbers needs to be inserted in the
    given order.
    Correct order: [NumberToBePowered] [^] [Exponent]
    Example: 2 ^ 2 will result into 4

Profiling

    An important part of the application is Profiling, that has been developed
    as separate application. In order to access profiling calculator, please
    open Menu section in the application Menu and click New Profiling Window.
  • What is Profiling application
    • Profiling is a function used to calculate selection standard deviations from
    sequence of numbers.
  • How to use Profiling application
    • Insert any amount of numbers from any text file, or put them into the input
    field. Data imported from files can be edited.
  • Correct form of data given.
    • Inserted data must be in given order. Separate all the numbers by
    whitespaces only. In order to make application work do not insert any
    other separators.

HotKeys

In order to make the application feel as comfortable as possible, we have integrated
options to use specific HotKeys.
  • Use your Digits or Numpad to insert numbers.
  • Use +,-,*,/ keys to call relevant operations.
  • Hit Enter or = symbol on keyboard to get result.
  • Hit F1 - F5 keys to call specific operations.
  • Bindings:
    • F1 -> Power Function
    F2 -> Sqrt Function
    F3 -> Logarithm
    F4 -> Absolute Value
    F5 -> Factorial
  • Use Delete or C to clear calculator display.
  • Use Backspace to delete number from backwards.
  • Use . or , to write comma.
  • Use CTRL + SHIFT+ P to open new Profiling Window.
